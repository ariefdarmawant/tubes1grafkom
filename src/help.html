How to use program

1. Pilih mode yang diinginkan

_______-Creating, untuk menggambar objek

_______-Editing, untuk mengedit objek

___Pada mode Editing dapat dipilih

_______-Recolor, untuk menganti warna objek

_______-Drag, untuk menggeser objek

_______-Resize, untuk menganti ukuran objek

2. Pilih bentuk yang diinginkan

_______-Lines, untuk menggambar garis

_______-Square, untuk menggambar persegi

_______-Polygon, untuk menggambar segi banyak

3. Pilih warna yang diinginkan dengan klik palate warna yang tersedia

4. Tombol save untuk menyimpan projek

5. Tombol load untuk membuka projek yang disimpan
